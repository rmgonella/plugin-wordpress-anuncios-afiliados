Oferta TOP Site - Ajustes 1.1.5

Ajuste solicitado:
- Adicionada a opção "Excluir" no menu administrativo Oferta TOP Site > Anúncios.
- A exclusão só pode ser executada por usuário com permissão administrativa manage_options.
- O link possui nonce de segurança do WordPress.
- Antes da exclusão, o administrador recebe confirmação no navegador.
- Ao excluir um anúncio, o plugin remove também registros operacionais de cliques e impressões vinculados.
- O histórico financeiro de afiliados é preservado, apenas desvinculando o ad_id do anúncio removido.

Observação:
- Esta opção não aparece para usuários comuns em "Meus Anúncios".
- O recurso foi mantido apenas no painel administrativo para evitar exclusões indevidas por anunciantes.
