Oferta TOP Site - Ajustes 1.1.3

Correções aplicadas:
- Corrigido botão Pagar agora para não reaproveitar links antigos de sandbox/teste do Mercado Pago.
- Ao clicar em Pagar agora, o plugin gera uma nova preferência de pagamento usando o Access Token salvo em Configurações.
- O plugin bloqueia/redireciona com aviso se o Mercado Pago retornar URL de sandbox, indicando token incorreto ou credencial de teste.
- Mantidas as credenciais de produção informadas nas configurações padrão.
- Mantidos saque mínimo de R$ 30,00 e ações de status de saque no admin.

Observação importante:
Depois de atualizar o plugin no WordPress, acesse Oferta do site TOP > Configurações, confirme se o Access Token começa com APP_USR e clique em Salvar configurações. Links de pagamento antigos já salvos no banco não devem ser usados; use o novo botão Pagar agora, que regenera o link em produção.
