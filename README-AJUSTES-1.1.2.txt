Oferta TOP Site - Ajustes versão 1.1.2

1. Admin > Saques
- Adicionada ação para marcar saque como Pago.
- Adicionada ação para voltar saque para Pendente.
- Adicionada ação para cancelar saque pendente, caso necessário.

2. Programa de Afiliados
- Saque mínimo configurado em R$ 30,00.
- Usuário só consegue solicitar saque com valor igual ou acima do mínimo.
- Campo de valor permite valores superiores ao mínimo, respeitando o saldo disponível.
- Formulário é bloqueado quando o saldo é menor que o mínimo.

3. Configurações
- Inseridos campos e valor padrão para credenciais de produção do Mercado Pago.
- Adicionado campo Valor mínimo para saque nas configurações do plugin.

4. Versão
- Plugin atualizado para 1.1.2.
